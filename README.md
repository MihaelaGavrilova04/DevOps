# BlushBox
Simple SpringBoot Test Project in Pink

**BlushBox** is a mini project for managing users and their documents, designed for **educational and demonstration purposes**.

The project uses **Spring Boot**, **Thymeleaf**, **CSS with a pink theme**, and a **database** to showcase how users can store, view, and manage their files.

Its main goal is **learning, experimentation, and demonstrating microservice architecture**, without handling any real sensitive data.

## Features

- Easy creation and viewing of users
- Add and manage documents
- Clean and visually appealing pink-themed interface
- Perfect for learning and testing

## Tech Stack

- Spring Boot
- Thymeleaf
- CSS with variables (:root)
- H2 / MySQL database (for demo purposes)
